# Uklady_rownan

1. Program rozwiązujący układ n równań liniowych z n niewiadomymi podany w wejsciowym pliku .txt w formacie:

n
A11 A12 A13 ... A1n B1
A21 A22 A23 ... A2n B2
A31 A32 A33 ... A3n B3
...   ...   ...   ...
An1 An2 An3 ... Ann Bn

gdzie: 
n- rozmiar układu równań 
Aij - macierz współczynników 
Bi - wektor wyrazów wolnych
